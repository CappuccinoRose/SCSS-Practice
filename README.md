# SCSS 语法练习项目

> 一个专注于 SCSS 核心语法特性的练习项目，通过实际案例掌握 @for、@each 等高级用法。

## 📖 项目简介

作为前端工程化的基础，CSS 预处理器是必不可少的知识点。本项目以 React 为载体，系统练习 SCSS 的常用语法特性，注重代码可维护性和模块化实践。

## ✨ 技术栈

- **React 19** - 前端框架
- **Vite 7** - 构建工具
- **React Router 7** - 路由管理
- **SCSS** - CSS 预处理器
- **ESLint** - 代码规范

## 练习内容

| 练习 | 语法 | 说明 |
|------|------|------|
| 练习一 | `@for` | 使用索引动态生成选择器和计算值 |
| 练习二 | `@each` | 遍历 Map 生成语义化样式类 |
| 练习三 | `@while` | 手动控制循环条件，适合复杂场景 |


## 学习笔记

### @for
适合已知循环次数的场景，如生成 `.item1` 到 `.item6`。

### @each
适合配置驱动的场景，如遍历颜色 Map 生成 `.item-green` 等语义化类名。

### @while
适合复杂条件判断，需手动初始化变量和更新变量，注意避免死循环。
